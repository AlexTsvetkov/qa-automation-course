# QA Automation Course

A comprehensive, beginner-friendly course for learning **test automation from scratch** using Python, Pytest, and Playwright.

## 📚 Course Structure

| # | Module | Lessons |
|---|--------|---------|
| 0 | [Cover & Table of Contents](course/00_cover_and_toc.md) | — |
| 1 | [Introduction](course/01_introduction.md) | Who this is for, what you'll learn, how to use the course |
| 2 | [Python Foundations](course/02_module1_python_foundations.md) | Variables, strings, lists, dictionaries, functions, classes (6 lessons + mini-project) |
| 3 | [Pytest](course/03_module2_pytest.md) | Test functions, assertions, fixtures, parametrize, markers (5 lessons + mini-project) |
| 4 | [Playwright](course/04_module3_playwright.md) | Browser automation, locators, actions, assertions, waits, screenshots (8 lessons + mini-project) |
| 5 | [E2E Testing with POM](course/05_module4_e2e_testing.md) | Page Object Model, base page, test structure, data-driven tests (5 lessons + mini-project) |
| 6 | [API Testing](course/06_module5_api_testing.md) | HTTP basics, GET/POST requests, response validation, integration (4 lessons + mini-project) |
| 7 | [AI-Powered Testing](course/07_module6_ai_testing.md) | ChatGPT for tests, Copilot, AI debugging, prompt engineering (4 lessons + practice) |
| 8 | [Git Basics](course/08_module7_git.md) | Init, commit, branches, merge, GitHub, pull requests (3 lessons) |
| 9 | [CI/CD Basics](course/09_module8_cicd.md) | GitHub Actions, automated test runs, reading CI results (3 lessons) |
| 10 | [Final Project](course/10_final_project.md) | Full test framework with grading rubric (100 points) |
| 11 | [Appendices](course/11_appendices.md) | Setup guide, cheat sheets, useful links, glossary |

## 🛠 Technologies

- **Python 3.11+** — programming language
- **Pytest** — testing framework
- **Playwright** — browser automation
- **Requests** — API testing
- **Git & GitHub** — version control
- **GitHub Actions** — CI/CD

## 🎯 Target Audience

Complete beginners who want to become QA Automation Engineers. No prior programming experience required.

## 🌐 Practice Application

All UI tests use [SauceDemo](https://www.saucedemo.com) — a free e-commerce demo site.  
All API tests use [JSONPlaceholder](https://jsonplaceholder.typicode.com) — a free fake REST API.

## 📖 Lesson Format

Every lesson follows the same structure:

- 🎯 **Goal** — what you will learn
- 💡 **Simple explanation** — concept explained with a real-life analogy
- 📋 **Code example** — working code you can copy and run
- ✅ **Expected result** — what you should see
- 📝 **Task** — practice exercise
- ⚠️ **Common mistakes** — pitfalls to avoid

## 🚀 Getting Started

1. Install Python 3.11+ ([setup guide](course/11_appendices.md#appendix-a--environment-setup-guide))
2. Install VS Code
3. Open [Module 1](course/02_module1_python_foundations.md) and follow along
