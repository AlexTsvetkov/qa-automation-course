# Introduction

## Who This Course Is For

This course is for **manual QA engineers** who want to learn test automation from scratch. You do not need any programming experience. If you can write a manual test case and click through a web application, you have everything you need to start.

## What You Will Be Able to Do After Completing It

By the end of this course, you will be able to:

- Write Python scripts to automate repetitive tasks
- Build automated browser tests using Playwright
- Create structured test suites with Pytest
- Design a Page Object Model framework
- Write API tests using the requests library
- Use AI tools to generate and improve test code
- Push your code to GitHub and run tests in CI/CD pipelines

## How to Use This Course

**Recommended pace:** 2–3 hours per day, 4–6 weeks total. Each lesson is designed to be completed in one sitting (30–45 minutes). Always type the code yourself — do not copy-paste. Making mistakes and fixing them is how you learn.
